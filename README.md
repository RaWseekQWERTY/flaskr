Anilog
A platform containing variuos Anime related Blog
Followed https://flask.palletsprojects.com/en/stable/tutorial/
<h2>Installation </h2>
<h3>Setting up environment </h3>
```sh 
python3 -m venv .venv
```
<br>
```sh
. .venv/bin/activate
```

<h2>Install Necessary Libraries<h2>
`pip install flask`

<h2>Installing the project</h2>
`pip install -e . `
